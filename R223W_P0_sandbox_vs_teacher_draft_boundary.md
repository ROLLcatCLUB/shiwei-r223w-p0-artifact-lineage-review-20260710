# R223W-P0 sandbox vs teacher draft boundary

## Teacher draft line

Purpose:

```text
Produce teacher-readable manuscripts that teachers can read, revise and use for preparation.
```

Canonical current sources:

- R223M_P4_P1_teacher_readable_process_v6.html
- R223N_P3_P1_teacher_manuscript_draft_v5.html
- R223O_P1_teacher_manuscript_draft_v2.html

Allowed work:

- improve lesson manuscript language
- improve classroom event expansion
- improve lesson structure
- improve teaching process and evidence

Where to do it:

```text
R223M / R223N / R223O manuscript line
```

## Sandbox / review shell line

Purpose:

```text
Review v0.1 / v0.2 candidate difference, review ledger completeness, component trigger safety and pilot gate readiness.
```

Current sources:

- R223T fixture-only sandbox preview
- R223U sandbox teacher review
- R223V sandbox reduction / pilot hold
- future R223W review-only pilot gate shell

Allowed work:

- compare v0.1 and v0.2 candidate
- show short difference summary
- show folded review ledger summary
- show safety flags
- show links/references to canonical teacher manuscripts

Not allowed:

- rewrite teacher manuscript body
- create new teacher draft baseline
- claim sandbox page is final teacher page
- add formal route or product UI

## Rule

```text
Teacher manuscript quality issue -> R223M/N/O
v0.2 schema or field issue -> R223P/Q
Sandbox readability issue -> R223T/W shell
Formal UI issue -> future R97B line, currently blocked
```
